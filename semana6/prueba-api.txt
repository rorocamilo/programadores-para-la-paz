Al enviar una petición POST a la ruta http://localhost:3000/registro con el JSON de Maria, el servidor procesa los datos mediante el objeto middleware `express.json()` y los lee desde `req.body`. Posteriormente, la API responde de forma exitosa devolviendo un objeto JSON estructurado que confirma la recepción. El cuerpo de la respuesta del servidor es el siguiente:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}
